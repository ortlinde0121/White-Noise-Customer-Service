# 白噪音客服中心 · White Noise Customer Service

欢迎致电异常客服中心。  
请保持在线。请不要相信来电者。请不要相信主管。请不要相信系统提示。

## 关于

《白噪音客服中心》是一款纯文字交互恐怖游戏。

没有画面，没有CG，也没有传统意义上的战斗系统。你将坐在客服工位前，接听一通又一通异常来电，在有限的选项中回应客户、处理事故、维持系统评分，并试着弄清楚自己究竟是谁。

这是一款独立短篇作品，与前作主线无直接关联。  
但其中包含两个来自前作角色的隐藏彩蛋。

## 故事

深夜的客服中心，总是会接到一些奇怪的电话。

比如抱怨冰箱在说话的独居老人。  
比如一直路过同一栋蓝色玻璃大楼的迷路女人。  
比如说电视里的人正在醒来的客户。  
比如听见门外传来规律敲门声的住户。

冰冷的座椅、劣质的塑料耳机、电流里不断循环的白噪音。

你是客服坐席04。

你的工作很简单：接听来电，安抚客户，判断风险，按照系统流程完成服务。

主管会批注你的回答。  
系统会评估你的表现。  
档案室会记录你留下的痕迹。

请继续接听。  
请保持专业。  
请不要主动询问客户是否为人类。

## 玩法

阅读来电内容，选择合适的客服回应。

不同选项会影响：

- 系统评分
- 主管信任
- 客户状态
- 档案收集
- 最终结局

部分特殊选项会留下 `SUPERVISOR-01` 的工作日志。  
收集足够日志后，可以在档案室解锁隐藏记录。

## 游玩时长

- 单周目约 20-30 分钟
- 多周目探索约 1-2 小时
- 全结局与全档案收集约 2 小时左右
- 档案室内提供辅助按钮，可减少重复刷流程

## 系统特色

| 系统 | 说明 |
| --- | --- |
| 来电处理 | 每通来电都有不同风险与回应方向，选择会影响后续记录 |
| 系统评分 | 系统会根据你的处理方式给予评分，并影响结局走向 |
| 主管信任 | 某些选择会改变主管对04的态度 |
| 档案室 | 通关后可查看来电档案、真实碎片、结局记录与隐藏记录 |
| 主管批注 | 特定选项会触发 `SUPERVISOR-01` 的私人工作日志 |
| 关于04 | 收集主管批注后解锁隐藏文本，补全04与主管之间的记录 |
| 环境音 | 内置由代码实时生成的白噪音、电话线杂音与电流故障声 |

## 收集内容

本作包含：

- 28 通普通来电
- 2 通隐藏来电
- 7 个真相碎片
- 4 个结局
- 18 条主管批注
- 1 份隐藏记录「关于04」
